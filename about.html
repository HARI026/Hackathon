Who We Are

This weather application is built to provide accurate and real-time weather forecasts for any city in the world. Our mission is to help users stay informed and prepared for weather conditions, no matter where they are.

Technologies Used

  • HTML, CSS, JavaScript for frontend
  • Responsive Design for all screen types
  • OpenWeatherMap API for real-time data
  • LocalStorage for storing city search history

Features

  • Real-time current weather
  • 2-Day forecast display
  • Search history saved and accessible
  • Animated and modern UI
  • Fully responsive across all screen sizes
